プライバシーポリシー・運営者情報

当サイトが取得する情報と、広告・アフィリエイトの取り扱いについて

運営者情報

当サイト「Claude Code 逆引き辞典」は個人 (GitHub: ry071702-prog) が運営する非公式のリファレンスサイトです Anthropic 社および Claude Code の公式サイトではありません お問い合わせは GitHub Issues までお願いします

取得する情報

  • 検索語の匿名統計 — 検索結果が0件だった語のみを、辞典の収録改善のため集計しています IPアドレス等の個人を特定できる情報とは紐付けていません
  • Skill 登録フォーム — 送信されたリポジトリ名・連絡先 (任意)・コメントを、掲載審査の目的にのみ利用します
  • localStorage — テーマ設定・お気に入り・講座の進捗を利用者の端末内にのみ保存します サーバーには送信されません

広告・アフィリエイトについて

当サイトは、第三者配信の広告サービス (Google AdSense) を利用する場合があります 広告配信には Cookie 等の識別子が使用され、利用者の興味に応じた広告が表示されることがあります パーソナライズ広告は Google の広告設定 から無効にできます

当サイトは Amazon アソシエイト・プログラムに参加する場合があります 参加時は、適格販売により当サイトが収入を得ることを該当箇所に表示します リンク先での購入情報が当サイトに共有されることはありません

免責事項

掲載内容は公式ドキュメント等を参照して正確を期していますが、その完全性・最新性を保証するものではありません 掲載情報の利用により生じたいかなる損害についても、運営者は責任を負いません コミュニティ Skill は第三者のコードであり、導入前に必ず中身を確認してください
